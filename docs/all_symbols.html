Array buffer to text, needed for crypto operations. The text is decoded from UTF-8, which is the default for the WebCrypto API.

Decode a string from encoded form. If the text is recognized as Multibase, the encoding format is there, otherwise the options argument rules.

Encode the result in base58 or base64, possibly in Multibase. The options argument dictates. Default is base64 encoding and plain output.
